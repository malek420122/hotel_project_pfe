# Frontend (Vue 3)

Interface cliente et administration pour la réservation hôtelière.

## Lancer en local
```bash
cp .env.example .env
npm install
npm run dev
```

## Fonctionnalités
- Navigation responsive (réservation, chambres, services).
- Support multilingue (FR/EN/AR).
- Vue Router + Pinia pour la navigation et l'état.
